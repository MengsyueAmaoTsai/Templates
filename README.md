# RichillCapital.Templates

Custom templates for personal project initialization and development standardization.

## Introduction

RichillCapital.Templates is a collection of project templates designed for personal use. These templates help developers standardize project initialization and accelerate the development process by providing predefined configurations, file structures, and sample code.

## Installation

```powershell
dotnet new install RichillCapital.Templates
```
